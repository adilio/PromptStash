# AGENTS.md

AGENTS.md is a community-standard file for AI coding agent instructions. It provides a consistent way to define system prompts across different tools and teams.

## Why it matters

As AI coding agents become more prevalent, having a standard format for instructions helps teams maintain consistency and share configurations across different tools.

## How PromptStash uses it

PromptStash can export prompts to AGENTS.md format, making it easy to manage your agent instructions in one place and sync them to your repositories.
